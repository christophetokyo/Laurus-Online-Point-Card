How to Add a Card Design

Step 1: Upload your card image to the cards folder on GitHub
(e.g. cards/warrior.png)

Step 2: Edit index.html on GitHub, find BUILTIN_DESIGNS near the top, and add one line:
warrior: { name: "Warrior", imgSrc: "cards/warrior.png" },
Step 3: Save and commit. Done!
